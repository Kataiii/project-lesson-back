Бекенд часть для веб-приложения, разработанная на Express и Typescript

Для запуска в корне проекта выполнить команды:

1. Установка зависимостей
   `npm install`

2. Запуск приложения
   `npm run dev`

Приложение запуститься на порту `http://localhost:5000`
Для подключения базы данных перейдите в файл `.env` и подставьте в переменные:

1. DB_NAME - название базы данных
2. DB_PASSWORD - пароль для подключения к базе данных

Документация swagger можно посмотреть по url: `http://localhost:5000/api-docs`
